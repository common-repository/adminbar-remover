== Description ==
Hide and show the adminbar by clicking WP logo. It works on front pages only. This plugin is basically for front-end developers who want to check design instantly.

== Installation ==
= From your WordPress dashboard =
1. Visit 'Plugins > Add New'
2. Search for 'AdminBar Remover'
3. Activate AdminBar Remover from your Plugins page.

= From WordPress.org =
1. Download 2D Tag Cloud.
2. Upload the 'adminbar-remover' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate AdminBar Remover from your Plugins page.
